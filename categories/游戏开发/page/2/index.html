游戏开发
为了测试《暗黑世界》客户端，所以搭建《暗黑世界》服务器端，特此记录共享！ 这个框架基于python编写，第三方库依赖：twisted， python-memcached，DBUtils，MySQLdb，后端数据存储用到了mysql和memcached。 那么咱们一步一步从头来，注意我这里是XP的系统。 1、安装python，我用的是python-2.7.5.msi，注意是2.7的版本，如果你装了2.6下面twisted和MySQL-python也就要安装对应的2.6版本，下载一路下一步安装即可，注意安装时选择把他的路径加到Path环境变量中，这样咱们就可以直接在控制台使用python命令了； python-2.7.5(windows_x86).msi 2、安装mysql，我用的是mysql-5.5.27-win32.msi，也是下一步安装即可，注意设置root账号密码，后面会用到，请务必记住密码，默认端口为3306； mysql-5.5.27-win32.msi 3、安装memcached，你可以下载源码回来这自己编译，也可以找别人编译好的，我找了一个1.4.13版本的：Memcached 1.4.13 For Windows 32or64.zip memcached.exe 关于memcached： 安装：运行cmd，进入memcached.exe所在路径，运行memcached.exe -d install -m 500 即可安装完毕，你可以去windows服务里查看是否有memcached服务； 启动：memcached.exe -d start 停止：memcached.exe -d stop memcached的基本设置： -p 监听的端口 默认端口：11211 -l 连接的IP地址, 默认是本机 -d start 启动memcached服务 -d restart 重起memcached服务 -d stop|shutdown 关闭正在运行的memcached服务 -d install 安装memcached服务 -d uninstall 卸载memcached服务 -u 以的身份运行 (仅在以root运行的时候有效) -m 最大内存使用，单位MB。默认64MB -M 内存耗尽时返回错误，而不是删除项 -c 最大同时连接数，默认是1024 -f 块大小增长因子，默认是1.25 -n 最小分配空间，key+value+flags默认是48 -h 显示帮助 4、安装setuptools，装这个其实就是方便使用easy_install来安装python第三方库，下载：setuptools-0.6c11.win32-py2.7.exe，安装完毕后，把python下Scripts路径也添加到path环境变量里，我这里是path：C:\Python27\Scripts； setuptools-0.6c11.win32-py2.7.exe 5、安装twisted，其实可以用easy_install twisted命令来安装，我这里是下载Twisted-12.2.0.win32-py2.7.exe来安装； Twisted-12.3.0.win32-py2.7.exe

孙婷采访

2005-03-28
游戏开发
“她是有丁香一样的颜色，丁香一样的芬芳。” 江南烟雨忆丁香 －－“唯晶宝贝”孙婷访谈录 第一次见到孙婷，是在去年的CHINAJOY展会上。那是记者暗金色月亮首次代表《消费电子世界》赴外地采访，偏偏遇上了首次以形象代言人身份出镜的“唯晶宝贝”孙婷。喜欢她不仅是因为她楚楚可怜的精致容貌，更因为她在展台前一站就是一天，始终保持职业笑容，对玩家没有一丝不耐的敬业精神。大半年过去，孙婷已戴上了“网游第一美少女”的耀眼光环，这朵已大放异彩的丁香，是否还固守着当初那淡淡的芬芳呢？ 关于Cosplay、代言人和游戏 暗金色月亮（以下简称月）：怎样走上代言人道路的？你心中的cosplay（即角色扮演，通过自己制作道具服装来模仿电影、电视、动漫、游戏中的各种人物）应该是什么样子？ 孙婷（以下简称婷）：刚开始我的一个女性朋友，到我家来装了一款网络游戏，带我一起玩，之后一发不可收拾，变成了游戏狂人……后来我觉得光玩游戏还不够，希望有更多途径接近游戏，甚至从事游戏类的工作。正好一些游戏开展了选秀活动，我就借此机会走上了自己梦想的舞台。后来机缘巧合，签约唯晶科技，正式接受了形象代言人的工作。 我觉得cosplay并不仅仅是单纯地穿漂亮衣服。大家都是因为热爱漫画、热爱游戏才幻想自己能够扮演其中的某个人物。至于如何把握游戏人物的神韵，我认为和个人气质有关，也有一点点天份的影响。例如我自己比较适合女性化的角色，因此特别偏爱公主类的cos秀。 月：玩过哪些游戏？最喜欢的游戏类型是什么？ 婷：我今天所拥有的很多东西都是游戏带给我的，所以游戏对我来说，算是半个生命。我最喜欢在不同类型的游戏中体验不同类型的角色。在《Dungeon Keeper》（《地下城守护者》）中我是一个大魔王，在《Baldur’s Gate》（《博得之门》）中我是游荡于奇幻的费伦大陆上的救世主，在《Fallout》（《辐射》）中我又成为拯救世界的神选者。每次玩RPG，都觉得好像在看自己演电影，随着情节忽悲忽喜，感受不同的人生体验。 月：代言公司的网游之后，会亲自去玩吗？到时候会不会公布自己所在的服务器和人物ID？ 婷：好的形象代言人必须了解代言的产品，所以当然要亲自体验游戏，否则就是欺骗玩家。将来游戏里的某个NPC可能就是我的作品哦！在游戏中我还是会用我经常用的名字，如果大家有兴趣知道更多的话，就多多关注我的动态吧。 关于恋爱、朋友和Fans 月：有男性玩家要我代问，目前有没有男朋友？心中好男孩的标准是怎样？ 婷：实话实说，目前还没有。就像一句大家都知道的台词里说的那样，“曾经有份真挚的感情放在我的面前，我没有好好珍惜”……将来我一定要牢牢抓住幸福。对于未来的他，我并没有一个明确的标准，因为好的男生各有各的优点，只要性格和得来就行了。不过我很希望他是一个孝顺的孩子，这样的人一定比较懂事、善良（握手，咱俩想到一块儿去了――月亮注）。 月：除了玩游戏还有别的爱好吗？ 婷：射手座的女生很厉害的哦！我喜欢骑马、卡丁车、唱歌、篮球、跳舞、滑雪、网球、钓鱼……当然最喜欢的还是吃（严重握手，好同志呀！――月亮注）。特长是书法，有7年书龄。对语言也颇有兴趣，目前会日语，还有一点点的葡萄牙语、法语和韩语。 月：家里人支持你的选择吗？身边的朋友有什么看法？ 婷：我从小就很独立，初中就开始盘算打工什么的，妈妈非常开明，基本上都会尊重我的想法，最多给我提些建议。朋友们刚开始当然会感到十分好奇，但大家还是和过去一样相处，时间长了也就习惯了。毕竟这只是一个职业，和医生、老师没什么区别，不过还比较新鲜而已。 月：作为偶像人物，有没有被疯狂的Fans吓到过？ 婷：我觉得不能用疯狂来形容啦，因为都是很可爱的人呀。每到过节的时候就有很多Fans送我礼物，真的很开心。最让我难忘的是我的第一个Fans，当时是我在CHINAJOY的最后一天，累得眼睛都睁不开了。那个男生一直坐在对过EA的展台上看我，陪了我4、5个小时。后来公司同事怕他对我图谋不轨，跑过去问他究竟想干嘛，他很不好意思地说，只是想和我握个手……还有类似在洗手间门口等我，追着我满场飞奔的行为，都很好玩。希望今年的CHINAJOY仍然能看到他们的身影。我在这里邀请大家了，看到这个访问的朋友记得再来看我！ 关于美丽、个性和生活 月：你觉得女玩家在网游中应该注意些什么？你对网游中的婚姻持什么看法？ 婷：女孩子玩游戏首先要注意游戏时间的长度。我刚开始玩游戏的时候由于太投入了，早上睁开眼睛开始玩，一直玩到第2天凌晨睡觉，醒了接着玩，脸色变得很憔悴，还长豆豆……那段时间都不太敢出门，后来还生病了，去医院挂着盐水回家，插着针头的手继续玩游戏。我这里提醒大家尽量控制好玩游戏的度。此外还要警惕网络里的一些坏人，学会保护自己。网游中的婚姻，因为存在一定的虚幻性，可以说是最单纯的爱情，也可以说是最不牢固的爱情。单纯是因为大家都没见过面，相爱纯粹是欣赏对方的性格。可一旦走到现实中来，会出现许多不定因素，所以说是不牢固的。我以前也有过这样的经历，游戏里的爱情投入过就好，不要太在乎结果如何。 月：美丽的女生常被人称作“花瓶”，你打算如何改变别人的这种成见？ 婷：我从来不认为自己是个“花瓶”，从上面我提到的兴趣爱好就可以说明我各方面发展得非常均衡，只是欠缺机会让大家了解。在普通人眼里，长得漂亮的多半没脑子，这是偏见！相信以后等我出了单曲，大家对我的整体印象会有所改观。这里再透露一个秘密，今年的CHINAJOY我将协助美术设计方面的工作，到时候欢迎大家来考察我的成果，我会用实力说话的。 月：学校里学到的知识对你从事代言人职业有帮助吗？这个职业又让你学到了什么？ 婷：我在学校主修服装设计和装饰设计，对自己的造型创意还是很有帮助的。去年的CHINAJOY就是我自己做的造型，虽然在网上被骂得很惨，不过是我第一次在自己身上做实验，相信今年一定会更好！至于现在这份工作，我从中学到了很多东西，毕竟我体验的可能是许多人一辈子也无法经历的感觉。在工作中我不断成长着。 月：生活中的你和舞台上有什么区别？ 婷：平面的照片容易给人安静的感觉，其实我在现实里是非常活泼的人，个性爽朗，完全符合射手座的性格特征，这一点相信见过我的人都有感触吧。 月：有什么特别的话要对女性玩家说吗？ 婷：希望所有女孩子都活得开心、快乐。自强自信是我们最有力的翅膀！

游戏开发
收到雷爵公司发来的媒体交流邀请函的时候，月亮以为这又是一次《万王之王2》的产品开发进度展示会。实际情况并非如此。尽管在公司一楼大批技术人员正紧锣密鼓地埋头苦干，二楼的会议室里探讨的话题却与这个游戏的内容本身没有太多联系。 “从2000年到2004年，中国玩家迅速成长起来了。” 暗金色月亮（以下简称月）：请问杨先生，为什么认为现在的网络游戏玩家已经发生了“质变”？对于“质变”你的定义是？ 杨瑞琨（以下简称杨）：我们可以看到，现在的网游市场，和当年《万王之王》（简称KOK）等第一批网络游戏刚刚兴起时的情形已经完全不同了。市场被打开后，玩家从刚接触这种新的游戏形式时的惊喜，慢慢转变为对产品同质化现象的麻木和厌恶，直至开始四处寻找一款真正适合自己的游戏。简单的说，我觉得他们经历了从无可选择到有太多选择再到理性选择的成长过程，我将这个过程定义为玩家的“质变”。 月：玩家的成熟是否是厂商成熟的必然结果呢？ 杨：不，我预言明年的网游市场将会非常混乱。2003年4月，中国还只有38家游戏公司，57款网络游戏。可现在呢？这个数字翻了3倍不止。想要入场分杯羹的国内外厂商越来越多，这里面必然会有不少极易破灭的泡沫。 “面对外资巨鳄，生存还是灭亡？这是个难题。” 月：雷爵在网游产品研发方面起步很早，应该不会惧怕来自国内外同行的竞争吧？ 杨：作为中国大陆第一款大型多人在线角色扮演类网络游戏，KOK曾创造过注册会员60万人的奇迹，这在当时的网络条件下是相当惊人的成就。但KOK的辉煌毕竟已成过去，我们需要一款作品重拾昔日荣光。从2001年8月，KOK走下坡路开始，我们就已经着手准备KOK2的产品策划。历时3年的制作，都是为了更有把握面对来自玩家的苛刻考验。 月：据说雷爵不打算自己运营KOK2？ 杨：在今年的CHINAJOY上大家都注意到了，一些国际游戏巨头纷纷表现出对中国市场的勃勃野心。他们进入中国，要么是直接并购现有的行业佼佼者，要么是独立成立技术资金实力雄厚的独资企业，总之一定会给目前的游戏企业带来很大冲击。即使有政策保护，一批尚在褓襁的中小企业也有可能遭遇灭顶之灾。 这样的情况下，雷爵希望能找到真诚的战略合作伙伴，互相扶持，共同进退，增强产品的核心竞争力。当年华彩在运营KOK时出现过一些决策失误，其中部分原因在于当时没有多少可资借鉴的成功经验。现在国内运营的各个环节上都涌现出不少经验丰富的强势企业，我相信强强联手能让我们在面对外资巨鳄时更加底气十足。 “和WoW、EQ2这样的作品比，我们最大的优势在于本土化。” 月：部分玩家始终认为，奇幻文化来自西方，也只有西方出产的网络游戏才能展现出奇幻文化的精髓，你怎么看？ 杨：网络游戏的内涵并不仅仅来自于它的文化背景。KOK之所以曾经吸引过那么多忠实玩家，是因为它的主体在于城邦，而城邦的主体在于人民，游戏中的一切事物都是玩家亲手创造。对于我们来说，提供给玩家一个创造自己历史的工具，比追究是否体现了奇幻文化背景，是否遵循了龙与地下城规则，要重要得多，也实在得多。 月：难道KOK2能完全无视WoW和EQ2的挑战？ 杨：我们起码有3点优势是这两款游戏所无法回避的。首先是目前的网吧和家庭硬件配置是否能支撑那样庞大复杂的游戏？其次，在运营成本上，我们起码能从架设服务器的成本方面节省出一大笔开支，从而降低游戏售价。最值得一提的是，中国人更了解中国人，前两款游戏是针对全球发行设计的，不可能根据中国国情进行太多调整。KOK2却完全可以跳出纯西方风格，探索更贴近玩家需求的中国模式。 “我们希望成为玩家最初和最终的选择。” 月：你预测将有多少KOK的老玩家会在KOK2公测时回来看看呢？ 杨：起码80％吧……毕竟这是他们的最初感动。而且我想他们会带着自己的新朋友一起回来。 月：期望值太高会不会成为一把双刃剑呢？ 杨：当然，这样的风险肯定存在。KOK2与KOK相比，上手门槛将会低很多。同时我们的职业不再有一转二转的概念，取而代之的是徽章系统。这个系统可以让玩家在技能选择和使用方面更加灵活，戴什么徽章就能拥有什么技能，变成什么职业，在国战中起到改变局势的关键作用。这些改变都将给KOK的老玩家带来惊喜。 月：玩家的“质变”将对雷爵公司产生怎样的影响？ 杨：在无数网络游戏间流浪的玩家也许很快就会发现，KOK系列是他们最初，也可能是最终，最好的选择。
