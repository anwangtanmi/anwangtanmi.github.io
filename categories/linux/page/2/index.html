本人经过一整天的折腾QAQ 就为了Ubuntu的两个最重要的驱动安装（无线网卡驱动+独立显卡驱动） 搞得心态都炸了好几次 神烦！！（(╯‵□′)╯︵┻━┻） 所以特地写一篇博文 把我期间遇到的一些问题进行总结 希望能让遇到同样问题的朋友少走一些弯路。 【最重要的一点！！！所以写在最开头！！！】 【一定要先进入BIOS，把secure root关闭！！！】 中文是 安全启动模式 记得一定要先关闭！！！ 1.Ubuntu安装卡死问题 在进入引导U盘以后，开始安装以前，有个选项是 *install Ubuntu，这里先不要进入，按“e”进入编辑，找到以linux开头的那行，将“quite splash – – -”改为“quite splash nomodeset”，然后按F10引导进入安装，就不会卡了。 2.登录卡死问题 双系统我安装完以后，是Ubuntu引导win10，在选择系统的界面，同理，Ubuntu，按“e”进去编辑，修改内容同上条。 3.无线网卡驱动 我的笔记本是暗影精灵3，我安装了Ubuntu以后，通过命令查看电脑的网卡设备，发现并不是百度说的那样，有eth0，wlan0这样的命名。比如如果命名是eth0，eth1，则eth0表示有线网卡，eth1表示无线网卡。【未完待续（等我整理完，再更新一些内容）】 查询命令参考：iwconfig，ifconfig 我的无线网卡型号是b822，这个型号的驱动文件Ubuntu已经有了，我关掉BIOS安全启动（secure boot），reboot以后WiFi就能用了。【未完待续（补查询型号的操作过程）】 4.NVIDIA显卡驱动 这个我遇到的最大的问题就是NVIDIA显卡驱动安装好以后和Ubuntu默认的nouveau驱动会冲突，导致重启的时候出现下图的问题，无法进入系统。【未完待续（补寻找显卡驱动版本以及安装过程）】 这里我也手打一下这个问题描述，便于遇到同样问题的朋友搜索。（我当时真的搜不到啊QAQ） acpi INT3400:00: Unsupported event [0x86] acpi INT3400:00: Unsupported event [0x86] acpi INT3400:00: Unsupported event [0x86] acpi INT3400:00: Unsupported event [0x86] acpi INT3400:00: Unsupported event [0x86] 解决方法：强制关机，在启动之前，选择系统的时候，找到“ Ubuntu ”，按“ e ”进入编辑，将“ quiet splash nomodeset ”修改成“ quiet splash acpi=off ”，按F10引导启动。

暗影精灵3 移动硬盘安装 Linux系统 最近在学Linux，之前虚拟机用着很不爽，正好笔记本换了个SSD，于是把换下的128G硬盘买了个硬盘盒搞了个移动硬盘，双硬盘+双系统 岂不是美滋滋。 一. 安装前资源下载 WIN10安装很简单，网上随便一搜结果多的是，这里就不叙述了。接下来说一下在安装Linux系统前我们需要做的一些准备工作（我选择的Linux是Ubuntu的）： 准备一个大于等于4G的硬盘； 下载Ubuntu镜像文件，我是去官网下载的：https://www.ubuntu.com/download/desktop； 下载UltraISO软碟通，官网：https://cn.ultraiso.net/xiazai.html； 下载DiskGenius，http://www.diskgenius.cn/download.php； 二. 安装前资源配置 制作U盘启动盘： （1）安装完成打开软碟通软件 （2）【文件】-【打开】，打开下载好的Ubuntu镜像文件 （3）选中用作启动盘的U盘 （4）【启动】-【写入硬盘映像】（5）然后[写入]就OK了 2. 接下来对移动硬盘进行处理： （1）打开DiskGenius。注：这软件在我电脑上经常卡顿，不知是我下载的版本有问题还是怎样，总之在处理过程中多多等待别心急并不会影响使用 （2）因为之前这张盘我是用来装系统的，还没格式化，这里先将它格式化，如果之前有分区，需要把分区都删除才能格式化 （3）看该硬盘是否为GUID格式，如果不是转一下 （4）然后对硬盘进行分区，这里我是这样分的：/boot = 2G， /swap = 8G， / = 50G， /home = 剩余空间，可以根据不同的需求来进行分区，当然最简单的还是分一个 / 和一个swap。具体的操作就是点【新建分区】然后在出来的对话框中修改大小以及格式，格式的话除了swap选择 [Linux swap…] ，其他的都设置成 [EXT4] 即可。GPT不用区分主分区之类的，所以那一项不用管。 因为我的硬盘全都分完了，所以接下来的图去官网随便找张做示范了。 3. 分区完成，准备工作就已经全部OK了，接下来开始安装。 三. 安装Linux系统 由于昨天安装系统时忘了拍照，且安装并不是很麻烦，这里根据记忆进行叙述： 将U盘与移动硬盘都连接到电脑上，开机并按 [F9]，选择从U 盘进行启动； 然后选择第二项 [ Install Ubuntu ] ; 开始是一些常规设置，如语言、完全安装还是精简安装之类的。要注意到了安装类型那一步，选择最下面的 【其他选项】； 因为做准备工作时已经将移动硬盘进行了分区了，接下来就是找到那几个分区，进行挂载。记得没错的话是每点到一个分区下方会有个【更改】选项，只需要在弹出的对话框中将【用途】那一栏选一下就OK了，4个分区都要配置一下（这里是因为我的分区方案是分为4个）； 可以看到下方还有个引导设备安装位置的选项，这里我选择的是移动硬盘（关于这个选择有点小问题，后面会提到）； 到了这里可以说系统就等于装好了，点击安装，接下来还有些常规设置，需要注意的可能也就是别设置了密码自己回头忘掉。 四. 你以为这就结束了吗？hhhhhhhhh 系统安装后所遇问题汇总： 当你安装完成，输好密码，然后画面卡住不动了，该怎么办？ 这里我遇到了这种情况，因为暗影精灵3是双显卡，自带的N卡跟Ubuntu自带的显卡驱动存在一些冲突，所以每每输入密码准备登录时都会卡住。 解决办法是在开机时选择recovery模式进入Ubuntu，然后参照这个帖子进行设置。https://blog.csdn.net/muyuxiaoxiang/article/details/80502703
