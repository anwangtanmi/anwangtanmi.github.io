打开 C:/Documents and Settings/你的用户名/Application Data/Microsoft/Templates文件夹，就可以看到Normal.dot模板文件了，删除Normal.dot模板，重新打开WORD！ 如果这样做了，重启之后，或者第二天又出现同样的情况，不能保存，另存问题，就可以判断是中毒了，主要是种了宏病毒，需要借助杀毒软件来查杀宏病毒。一般的杀毒软件都可以杀，不过要在设置里面选中检测宏病毒（有的杀毒软件不是默认会杀宏病毒的），杀毒完成之后再按上面的步骤做一遍，就可以了。 希望大家来报告各种问题。谢谢

robots.txt

未分类
robots.txt是一个纯文本文件，在这个文件中网站管理者可以声明该网站中不想被robots访问的部分，或者指定搜索引擎只收录指定的内容。 当一个搜索机器人（有的叫搜索蜘蛛）访问一个站点时，它会首先检查该站点根目录下是否存在robots.txt，如果存在，搜索机器人就会按照该文件中的内容来确定访问的范围；如果该文件不存在，那么搜索机器人就沿着链接抓取。 robots.txt必须放置在一个站点的根目录下，而且文件名必须全部小写。 robots.txt写法 我们来看一个robots.txt范例：http://www.w3.org/robots.txt 访问以上具体地址，我们可以看到robots.txt的具体内容如下： # # robots.txt for http://www.w3.org/ # # $Id: robots.txt,v 1.48 2007/10/16 05:31:15 gerald Exp $ # # For use by search.w3.org User-agent: W3C-gsa Disallow: /Out-Of-Date User-agent: W3T_SE Disallow: /Out-Of-Date User-agent: Mozilla/4.0 (compatible; MSIE 6.0; Windows NT; MS Search 4.0 Robot) Disallow: / # W3C Link checker User-agent: W3C-checklink Disallow: # exclude some access-controlled areas User-agent: * Disallow: /2004/ontaria/basic Disallow: /Team Disallow: /Project Disallow: /Web

威盛暗助OQO入华

11月27日，OQO来到中国，在京签订了开拓中国市场的战略协议。不过，OQO的进入显得有点草率，其总裁首席执行官透露：“这仅仅是我第二次来到中国。” 比特网(ChinaByte)12月3日综述(王培垠) 移动生活方式在全球方兴未艾，超便携式个人电脑市场成为众多企业期望的下一个数千亿美元产业，整个产业从上游芯片设计到下游渠道企业都对其充满期望，在等待这个市场成熟的过程中，为超便携式个人计算提出的UMPC、UMD这些时髦的概念产品名称，已经开始走向不同的结局。 OQO首进中国却无心插柳 今年年初的CES上，美商OQO的确得到了比尔·盖茨的推荐，OQO的产品也成为全球能够运行Vista的最小个人电脑。从此，超便携电脑产品便让所有人眼红。这个定位于生产可以放在上衣口袋中的电脑的企业，在美国市场已经拥有相当影响力。 11月27日，OQO来到中国，在北京与世纪奥通科技有限公司签订了开拓中国市场的战略协议。不过，OQO的进入显得有点草率，其总裁首席执行官Dennis Moore透露：“这仅仅是我第二次来到中国。” 显然，OQO还没有将中国这个许多企业认为是金砖四国的新兴市场作为最重要的增长来源。相比中国文化和美丽的风景，中国电脑市场对Dennis Moore的吸引力还稍逊一筹。 作为超便携电脑的倡导者，OQO的用户群在欧美等成熟的商务市场才刚刚开始形成。对中国市场的陌生让OQO并没有选择在中国设立分支机构等许多国际企业惯行的方式，那么OQO更不可能成立中国区建立自己的直接管理的市场营销队伍，仅仅选择与渠道企业的合作，无疑显示OQO其实还并不认可中国市场目前对超便携设备的实际需求能力。 作为OQO的竞争对手，索尼、三星等国际消费电子的巨头，都曾在英特尔的大力支持下，发布过各自的超便携个人电脑产品，然而市场终究观望者居多，掏腰包的人少，超便携电脑尚没有给这样的国际巨头惊喜。对于OQO来说，中国市场最多也只是带去锦上添花的效果。 从三星的超便携产品Q1来看，三星不仅在中国拥有成熟的渠道和分销，其国代派普也在鼎立支持，很早就开始针对其产品进行策划研究和市场铺垫。但是派普从渠道的信息反馈看来，由于超便携电脑高昂的价格和缺乏杀手级的应用，超便携电脑尚难对传统的笔记本或台式电脑形成威胁，这不仅是三星需要面对的现实，英特尔作为个人电脑产业的推动者也不得不反省自己。而作为英特尔的下游合作伙伴，索尼、三星等目前都在为一个问题犹豫：整体产业环境对超便携个人电脑的支撑，能否达到使其大规模普及的程度? 所以现在OQO进入中国市场，既不看重直接在收入上能够取得进展，也无意设立中国分支提前铺垫市场，那么推动OQO进入中国的不排除有其他力量推动，而没有多少OQO自愿的因素在其中。 威盛想在英特尔身后开火 在此次OQO来到中国的背后，隐藏了另一个熟悉企业的身影：威盛。 在OQO与世纪奥通签约的当天，威盛中国区PC平台业务部高管黄羲家来到现场，并接受媒体专访。其高调的出现，甚至超过了OQO本身，然而OQO并不介意。 威盛对于超便携个人电脑的态度相当暧昧，既想扩宽自己的市场空间，又想避开与英特尔的直接对抗。原因在于x86架构下的个人电脑市场，威盛虽然也有相应的产品，但处于弱势地位，不得不剑走偏锋，避开与英特尔正面竞争，走上了高集成化和低功耗的道路。 过去，威盛更多针对的是工业用途的行业市场，但是低功耗处理器和高度集成化的设计理念向移动市场迁移后，威盛想要找到自己的出路不得不面临与英特尔撞车的局面，威盛也就必须放手一搏。 之前，瘦客户机、低端的一体机和笔记本都还能看到威盛的影子，但是威盛的许多合作伙伴都非常小心的与其展开合作。在桌面计算市场，与OEM企业合作的决定权始终在英特尔手中，过去许多下游企业在市场上能够以较大的竞争优势出现，必然得到了英特尔的默许和支持。 英特尔同样看重超便携市场，威盛很难从英特尔的眼皮底下打开缺口。当英特尔推出UMPC这个概念的时候，威盛从自身的产品市场中发展出UMD的概念。 威盛认为，英特尔可能更多的立足于个人电脑的市场，而威盛面向的更多是包括超便携个人电脑市场在内的整个便携式终端市场，还包括手机终端、GPS或者PMP等设备，这似乎令威盛找到了说服合作伙伴的理由。 的确，威盛看到自己的处理器在多普达、OQO这些企业便携式产品上的应用，也小心的避免与英特尔撞车。然而随着移动互联网的发展，威盛处理器所处的这些领域都在不断的被英特尔所侵蚀。虽然三星等个人电脑企业在其超便携个人电脑初期仍在使用威盛的处理器，但目前威盛的战友似乎又要像过去在台式机笔记本领域一样被英特尔抢走。 面对这种情况，与其坐以待毙，不如同样通过联合合作伙伴，推动市场成熟而抢先赚取利益，试想除此之外，威盛是否还有第二条路可行。目前，英特尔的处理器已经进化到45nm，在计算性能和功耗控制上都上了新台阶，更适合在移动便携式市场使用。威盛除了动用自己仅有的能力，拉动OQO这样的合作伙伴去抢夺自己的市场，别无选择。 作为源自台湾的企业威盛更看好中国的市场潜力，即便现在便携式移动计算市场还非常不成熟。OQO作为威盛的铁杆，让其走进中国市场，这是威盛希望看到的结果。 UMD面临困境 比尔·盖茨的个人推荐让OQO大放异彩，但是一家OQO在电脑产业中显然无法支持威盛的梦想。缺乏个人电脑企业的支持，使得威盛又面临一个尴尬的境地。当英特尔推陈出新，放出MID概念时，威盛提倡的UMD产品陷入比较尴尬的境地。 作为个人电脑领袖企业的英特尔，对于UMPC的淡忘，让威盛有些不知所措。英特尔已经基本放弃宣扬其UMPC的提法，转而到处推销MID的概念。 原因一方面在于，UMPC价格昂贵，并始终缺乏杀手级的应用和定位，以个人电脑的应用市场无法体现UMPC这种超便携产品的差异化，所以消费者难以接受。同时，英特尔提炼了便携式移动市场中的关键应用，即通过无线网络链接互联网，从而丰富MID设备的应用，也与传统笔记本或者台式机的定位有所区别。 英特尔认为，MID主要实现的是在足够便携的产品中，无缝、无差异的实现目前互联网的所有应用。而MID上的应用实现需要让用户获得相同的体验，和平台无关。这些更为精准的定位，使得英特尔的MID产品不会与现有的产品线产生冲突，又能够给消费者更为准确的应用远景。 反观威盛依然在推动其粗糙的UMD概念，既然都是一种便携式电脑，那么消费者用什么样的理由说服自己购买一台威盛的UMD呢?虽然威盛的UMD抓住了未来的方向，但是缺乏与英特尔相当的细分市场能力和产业协同合作的能力，威盛和OQO都显得有些孤单。UMD主导超便携移动计算市场还是一个梦想。 另外，威盛的UMD不仅受限于个人终端设备的小型化、低功耗的发展，而且在移动的条件下更受到无线网络的约束。而英特尔在WiMax等无线城域网标准上，则是主要的推动者。 英特尔看重的是，未来真实需要的是互联网，而不是一台个人电脑。通过哪种终端设备接入互联网并不重要，重要的是获得和过去在个人电脑上一样的互联网体验。 现在威盛UMD阵营中的合作伙伴，在缺乏强大的产业上下游运作能力和内容、网络应用的支持下，OQO们可以很快的转变为英特尔优秀的下游伙伴。OQO今天来到中国很难帮助威盛敲开超便携计算市场的大门，在移动互联网应用MID的强大攻击下，UMD真的很难。

除了用假链和死链欺骗网民的小网站之外，稍微有点职业道德的大型网站也往往会将下载链接转移到另一个页面，这里拥有不同服务器选择的界面，选择了服务器后才能在下一个页面找到下载链接。PCHome等拥有多台服务器的大型网站便采取这种做法，这样虽然能够成功地找到自己想要的资源，但其间须要点击重重链接，谈不上任何用户体验。一些小型网站更是让用户在多页面中跳来跳去，最后给出的还是死链、假链甚至其他网址。而另一些网站则通过让用户注册并登录或者须要在论坛回复获得密码后才能得到下载链接，这和多页面跳转异曲同工。 按照网站的人性化设计标准，用户应该最多通过三次点击得到一条有用资讯。那为什么还有许多站长热衷于将网站做得更不人性化呢？这其中无非一个“利”字在作怪！一个网站的价值基本体现在访问量上，无论网站的内容如何，只要用户在站内停留的时间越长，点击的数量越多，“每用户访问页面数”就会提高，整体访问量也会进一步提高。多页面的组合和用户注册虽然并不会让用户真正关注这个网站，却能够迫使用户访问更多的页面，最大限度地骗取每位用户的点击率。同样的，过期页面和假链也因此存在于网站上，部分网站并不将过期链接和过期页面修正和取掉，就是因为它们依然能够为网站带来访问量。所以，当你一而再再而三地遇到死链接，你还会认为这只是你的运气不好吗？ 除了资源性站点以外，大多数商业网站还是以新闻和资讯为主。在网络开始流行初期，网民们的信息主要来源于几个门户网站，现在网站的数量随着IT业的发展逐渐增加，网民们的选择渐渐多了起来，网站之间竞争也同时进入了一个怪圈。 网站之间互相竞争对网民来说本应是有百利而无一害的，资讯网站之间的竞争无非是围绕着网站的内容做文章。无论是增加原创能力还是增加资讯整合能力，都会让网民们看到更多的原创文章或者形式更好看的专题性文章。但是由于网络信息的特殊性，文章复制几乎零成本，而原创一篇文章却须要花相当的心思。再加上现在版权保护体制不完善，一篇优秀的文章往往被其他网站转载，原创文章得不到很好的保护。长此以往，更多的网站将工作重心倾斜到转载与文章包装上，而不愿意再进行原创了。 起初原创文章被转载，网站的工作者还会据理力争，而另一方则拒不认错。可是此类的争吵和叫骂由于难以获得有效的法律支持而不了了之，成熟起来的网站建设者们明白了争吵是无谓的。渐渐的，网站之间也形成了默契，大家互相转载对方的文章——无论是以签转载协议的方式还是默许，只要求注明出处。这样既在道义上维护了各自的权益，又在网站利益上获得一个平衡点，有些网站甚至欢迎转载以求宣传自己的站点。互相转载的方式粗略看来似乎是皆大欢喜没有什么问题，但这样却让网民们无处可逃，好像待在一个克隆岛上。大多文章千篇一律，同一篇新闻被改成各种标题分布在不同的网站之中，而各类网站也不再有特色可言。 用暴露的画面和诱惑性语句吸引用户点击，对于有自制能力的成年人来说最严重的后果也不过是受到欺骗而已，但是对于占据网民总数16.4％的未成年人来说，却可能造成不良的兴趣导向甚至走向歧途。由于网络环境的特殊性，对内容的监管相比传统媒体难度更高，虽然国家一直致力于及时取缔违法的色情网站，但是类似这种不算触犯法规，又在违规边缘游走的网站，在法规与道德的夹缝之间依然能够存活下来，甚至还活得很滋润。 网站赚钱的畸形之路 网站如何推广，如何盈利一直是IT工作者从“.com时代”就奋力追求的目标。经历了从服务收费、广告、短信等不同阶段不同方式的赚钱途径洗炼之后，网站从业者们发现无论是何种方式的盈利，最终所需要的条件都是一个数据“访问量”，访问量所决定的Alexa排名和Google的RP值能够让更多访问者知道并来到网站。就这样，访问量就成了网站站长们每天最关心的数据，为了让更多的网虫知道自己，一些正规技巧的和更多投机的方法在网站制作者之间流传开来。 ◆给搜索引擎“浏览”的网页 在网页设计的投机技巧中，一般都是针对搜索引擎的机器人对网页作出特殊调整，以便让网页更加容易被搜索引擎找到并收录。其中最典型的便是“鬼页”。所谓“鬼页”，是指网友看不见、然而搜索引擎能检索到的网页。其目的是通过增加标题、网页简述、网页正文中的关键词曝光率，提高网站的“关联度”，从而使网站的排名更加提高。具体的做法是将关键字的文字颜色和背景颜色做成一致或者接近，以至于让人眼无法分辨。当一个页面全部都是这种无法看见的关键字组成的时，这个页面就被称为“鬼页”。一般在制作“鬼页”时并不会将这个页面完全做成鬼页，而是会在页面上方放一些可见的内容，而下方则用“鬼页”的形式堆砌关键字，以欺骗搜索引擎机器人。 除了“鬼页”之外，还有其他的一些类似的方法欺骗搜索引擎。比如将关键字写进图片的说明里，一般方法是建立一张一个像素的图片，并且在IMG标签之后加入Alt属性。因为搜索引擎对链接图片的Alt属性权重非常高，所以Alt里所写的关键字就很容易被搜索找到，但是在用户所看到网页里却根本看不到与这些关键字相关的内容。另外还有在网页注释中添加关键字的方法同样是欺骗搜索引擎的手段。以上的方法在现在大多数的搜索引擎中都已经能够被辨识出来，并且多数搜索引擎也会将这些类型的作弊方式屏蔽掉，但是依然有部分搜索引擎无法将其全部屏蔽。 图8：在网页源代码的注释中加入关键字检索让搜索引擎获取 ◆用技术“化妆”，讨搜索引擎“欢心” 除了在关键字上欺骗搜索引擎之外，一些网站还喜欢在网站结构上下工夫。比如将一个页面分拆成多个页面，化整为零，增加搜索引擎检索频率，从而优化排位结果。这种方法被叫做“桥页法”（Bridge pages）。和正常的分页不同，桥页法是把一个完整的页面按各自的区域分别裁剪成许多独立的网页，每个网页都有自己的Title，URL以及关键字。桥页法的好处是能够让网站的关键字密度大大提高，而且一般不会给用户带来不便并且也不会受到搜索引擎的反感。还有些网站上使用META刷新标记、CGI程序、Java script或其他技术，让用户点击一个链接时，除了转到链接的网址外，还会打开一个新窗口跳到另一个网页。虽然在有时候这种功能是必须的，但是有些网站使用这种方法欺骗网民进入并不想去的地方，进而增加用户页面访问量。而那些将某个网站排名和访问量提高后再演一出狸猫换太子的把戏，将网站换成其他页面的方法就更加让网民苦不堪言。 后话 资源下载大唱“空城计”，新闻资讯换汤不换药，搜索引擎挂羊头卖狗肉，门户网站倚门卖笑，中小网站更是歪门邪道鸡鸣狗盗……除此之外我们还要忍受形式各异极占系统资源的Flash广告，还要防毒防骗防木马，说严重点上网就像在上战场。难怪有人认为中文网络已经垮掉了——被流量胁持，在用户可以忍受的底线范围内追求利益最大化，最大限度地浪费网络资源以及网民的时间。 俗话说：君子爱财，取之有道。任何急功近利的“短视”行为都只会将网络引向歧途。从以前的网站烧钱做内容，到现在的赚钱做流量，每个阶段都见证着中文网络的发展和成熟。但是，过犹不及，在中文网络万紫千红遍地开花的时候，我们不得不警钟长鸣：树立一个科学的发展观，节约我们的网络资源，还网民一个干净清澈的网络空间吧！ 本文刊载于《计算机应用文摘》2005年第22期
