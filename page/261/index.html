洋葱致西瓜——三个月纪念

2004-11-16
西瓜和洋葱的故事是这样开始的。洋葱本来就是西瓜身边的一只大尾巴狼，表面上和善可亲，事实上心里总是打着自己的小算盘。恋爱这种事情，男生总是主动些，所以有一天大尾巴狼找了一个借口和西瓜吃饭。这一顿饭，最终成为了一场鸿门宴。饭局里，洋葱问西瓜：“你觉得我追你怎么样？”也许是因为觉得这一问题非常没有层次，问法也不够讲究，没有人愿意回答这个问题。当时的气氛很尴尬，没人知道说什么好，饭没吃完（每次和西瓜吃正餐都是要剩下东西的），两人就匆匆离开了。故事到这里似乎就要结束了，但是没有波折怎么叫好故事呢。西瓜说要逛街。后来的资料表明接下来所发生的事情不是逛街而是徒步竞走，类似长征。两人从武汉大学走到华师，再到虎泉，然后又走回来（虽然途中西瓜很累，但是坚持不搭taxi。西瓜是很有节俭原则的人。这一点俺颇为欣赏。西瓜从来不卖贵的东西。比如买手机，价格高了，她就不要了。但是不要被假象所迷惑。在西瓜的字典里，节俭的意思是这样的，比如衣服，绝对不买太贵的衣服，比如两百元以上的，但是200元以下的呢，决不手软）。途中洋葱把自己脑子里的陈芝麻烂谷子的故事全都弄出来了，还讲了一个伟大的诺贝尔得主的故事。可惜洋葱的故事大多是哲学家和数学家的，这些都超出了西瓜小脑袋的容量。更何况洋葱小学换了5个语文老师（免责声明：老师的更换和洋葱本人无关），表达能力极其有限，所以你可以经常那个看见在洋葱发表一番长篇大论之后，西瓜满脸轻松的说道，不懂。但是后来的事实证明这次徒步行走的意义是非常巨大的，它催生了洋葱和西瓜伟大友谊的萌芽，为今后友谊的发展奠定了坚实的基石。据考证，这一天是在2004年7月28日。（时间，意义都要记住，是要考试的啊） 接下来的日子是非常难熬的。这种感觉就像胸中煮了一锅开水，指不定什么时候就沸腾一下。有时候洋葱一整中午都睡不着，就像今天一样。但是，西瓜是一个慢性子。急性子的洋葱最终决定让西瓜作出明确的选择。这一天是8月16日，在奥场，漆黑的夜伸手不见五指。两人像时针一样围着操场转啊转啊。在洋葱巨大的心理攻势之下，西瓜终于决定接受洋葱的伟大友谊。 8月22日，七夕。洋葱想送西瓜一件礼物。没有经验，所以决定送花，并且想好了一个非常得意地台词。但是后来的事实证明洋葱的确是理论的高人实践的矮子。洋葱的计划是这样的：他打算送11支百合给西瓜，但是只买10只。当西瓜发现只有十支的迷惑不解的时候，洋葱及时出现深情地对西瓜说，你就是其中最美丽的一支。镜头最后对准西瓜幸福的一塌糊涂的脸。可悲的是，洋葱不知道百合不是一支只有一朵花（早知道就不买百合买玫瑰了）。真实出现的情景是，西瓜数出来的居然比11朵还多。失败！！！！！ 洋葱语录：世界已经无可避免的走向庸俗，让我们来拯救最后的浪漫。（西瓜，记得不，8月21日的晚上。俺们还谈了佛和一棵树。） 西瓜语录：1）不过幸亏他碰到的不是一个太笨的女生，所以他不用说什么，他想说的她都知道！（04年9月4日21点01分，那一天西瓜给别人帮忙忙到很晚（西瓜就是不会拒绝别人）。第二天还要上课。洋葱有点不高兴（他老是为芝麻大的事不高兴）。为了上课，西瓜很晚很晚依然回家。洋葱是一个很笨很笨的人，总是语无伦次的表达自己，或者更干脆的，找不到词语表达自己，所以西瓜说了上面的话。） 2）哎，我刚看到一句话——遇上第一次谈恋爱的男生是一件很可怕的事情。那是一个固执小气，大惊小怪的男孩子，只会给你添麻烦。呵呵！这句话是张小娴说的。说得似乎没错。（04年9月28日14点59分，那时洋葱在回家的火车上。这句话一点没错，这几个月来，俺们一共吵了9次架，一次比一次厉害。应该不能说是吵架，因为西瓜从来不和我吵架。我知道那个时候为了忍受俺的脾气，西瓜一定受了不少委屈，因为西瓜也是一个脾气很倔的小公主，但是她从来没有和我吵过。她说，你喜欢我才会这样紧张我，所以我为什么要和你吵（不过西瓜也是个理论高手哦）。可悲的是，我后来得寸进尺了，不提也罢。西瓜大人多包涵了。） 3）不管怎么样，你觉得有问题肯定是我做得不对或者是某方面做得不够好，所以我总是道歉。我确实每次都一直在做自我检讨，不希望下次出现同样的事让你不开心。我是女生。每个女生都希望被自己的男朋友宠着惯着，开开心心的！但是我为什么感觉不到呢？我从不怀疑你对我的感情，我只是觉得你爱的方式让我很有压力，因为有压力而快乐不起来，其他的更无从体会!累，是我很真实的感受。这不是我想要的爱情！（04年9月30日23点48分，这一天我们又一次发短信发到一点多钟，只是为了我闹脾气。我会把这条短信永远留着。我以为我会是一个好男朋友，但是我不是。西瓜，谢谢你的容忍，对不起（有点酸酸的）。另外，需要指出的是这句话也充分体现了西瓜的文字功底。请注意“不希望下次出现同样的事让你不开心”这句话。这里的“希望”用的非常的贴切，也可以用“想”来代替。西瓜本人对这一神来之笔亦非常满意。满脑子小聪明的瓜瓜！） 4）对不起，现在有事，稍候联系！^.^（西瓜专用短信，不过已经被我成功盗版了，但是不敢用。） 昨天晚上，西瓜问我，知道明天是什么日子吗？笨西瓜，我当然知道，我们一起走过三个月了。三个月里，我们之间发生了太多太多的事情。有洋葱第一次吃肯德基，第一次吃热干面，也有两个人吵架，洋葱半夜搭taxi出门，甚至说分手。很多不开心的事，都是因为我的大惊小怪。如果不是西瓜，我不知道会是怎样的结局。西瓜，今天是我们三个月的纪念日。我总觉得自己欠着你些什么，所以中午睡觉的时候老是睡不着。西瓜，今天我不无法在你身边，无法给你礼物，只能在这冰冷的屏幕前敲打键盘来叙述自己的感情。西瓜，我爱你!我们三个月我们就不要纪念了，等到第一个一万年的时候我们再庆祝，好吗？ 天天面对电脑怎么保健

2004-11-09
未分类
目前，电脑对人体生理和心理方面的负面影响已日益受到人们的重视。为此科学使用电脑，减少电脑和网络的危害是十分必要的。 一是要增强自我保健意识 工作间隙注意适当休息，一般来说，电脑操作人员在连续工作1小时后应该休息10分钟左右。并且最好到操作室之外活动活动手脚与躯干。平时要加强体育锻炼，增强体能，要定期进行身体检查和自我心理测定。 二是注意工作环境 电脑室内光线要适宜，不可过亮或过暗，避免光线直接照射在荧光屏上而产生干扰光线，工作室要保持通风干爽，能使那些有害气体尽快排出，尽量用非击打式打印机减少噪音。 三是注意正确的操作姿势 应将电脑屏幕中心位置安装在与操作者胸部同一水平线上，最好使用可以调节高低的椅子。坐着时应有足够的空间伸放双脚，不要交叉双脚，以免影响血液循环。 四是注意保护视力 要避免长时间连续操作电脑，注意中间休息。要保持一个最适当的姿势，眼睛与屏幕的距离应在40－50厘米，使双眼平视或轻度向下注视荧光屏，这样可使颈部肌肉轻松，并使眼球暴露面积减小到最低。如果出现眼睛干涩、发红，有灼热或有异物感，眼皮沉重，看东西模糊，甚至出现眼球胀痛或头痛，那就需要到医院看眼科医生了。 五是注意补充营养 电脑操作者在荧光屏前工作时间过长，视网膜上的视紫红质会被消耗掉，而视紫红质主要由维生素A合成。因此，电脑操作者应多吃些胡萝卜、白菜、豆芽、豆腐、红枣、橘子以及牛奶、鸡蛋、动物肝脏、瘦肉等食物，以补充人体内维生素A和蛋白质。而多饮些茶，茶叶中的茶多酚等活性物质会有利于吸收与抵抗放射性物质。 六是注意保持皮肤清洁 电脑荧光屏表面存在着大量静电，其集聚的灰尘可转射到脸部和手的皮肤裸露处，时间久了，易发生斑疹、色素沉着，严重者甚至会引起皮肤病变等。 生活时报

便秘的饮食治疗

2004-09-19
合理饮食是治疗便秘的良方。 一、多吃富含粗纤维的蔬菜和水果。如芹菜、韭菜、菠菜、桔子、香蕉等。粗纤维可刺激肠壁使肠蠕动加快。 二、多饮水及果汁软化粪便。晨空腹饮加少量食盐的温开水300毫升，其效极佳。 三、炒菜时，适量加入烹调油，可起到润滑肠道之功效。其中的脂肪酸兼有促进肠蠕动之功能，从而使肠道润滑、畅通无阻。 四、多食富含维生素B的食物。如粗粮、酵母、豆类、洋葱头、萝卜等。 五、各类干豆。肠道内的正常细菌可使其发酵、产气，促进肠蠕动。 六、多吃润肠通便的食物。如银耳、蜂蜜、香蕉等。 便秘者应禁忌饮酒、喝浓茶、喝咖啡，忌吃辣椒等刺激性食物。 点赞 收藏 分享 文章举报 djt2000 发布了63 篇原创文章 · 获赞 0 · 访问量 9万+ 私信 关注

非常经典的像素画系列教程–第 7 页 明暗关系 在像素画中仅仅把握住了明快的色调还不够，有时画面的色彩仍然不够生动，这是因为在色彩的运用还缺少明暗的变化。 只有遵循物体与光源之间的变化关系，才能表现出既生动又符合我们视觉印象的像素画来。 当我们准备绘制一幅具有明暗的变化关系的像素作品时，首先要定好光源的所在位置，其次根据光源的照射再在物体上添加亮部、中间色、明暗交界线、反光、投影，如果绘制的像素作品很微小的话，那就只要把握好亮部与暗部就可以了。 物体的明暗刻画 如图所示，在A图的基础上考虑进去光源照射的方向并且把握好物体的受光面与背光面。其中应当注意的是物体无论是受光部分还是背光部分的色调要与整体色调保持一致，亮部可以由主色加白色或浅色提亮，而暗部可以由主色加黑色或深色转暗。 人物的明暗刻画 如图所示，比物体的明暗刻画要来的复杂，虽然受光照射的方向一样，但是人的头发、帽子、皮肤、穿戴的色调以及物体质感皆有所区别，因此我们还是应该根据物体属性分别来刻画它们。如果有背景的画，有时还要考虑一下背景对人物的反光，也就是我们说的环境色。 重叠物体的明暗刻画 如图所示，两个或者更多物体重叠在一起时，除了要遵循明暗的变化规律，而且还要注意物体与物体之间重叠时产生的叠影，这样物体之间将更具真实感。

Game API 通过下面的四个类提供了灵活的对层的支持: Layer 是所有层的抽象父类. 它定义了层的基本属性,包括位置，大小以及是否可见. 每个Layer 的子类必须定义paint() 方法用来在Graphics 上绘制该层. 它的两个具体的子类, TiledLayer 和 Sprite, 应该会满足对2D游戏的需求. TiledLayer 在创建背景图时会被用到. 你可以用一组小块儿的图片来高效地创建大幅的背景图. Sprite 是会动的层. 你提供每个动作(帧)的图片然后对它的活动进行完全的控制. Sprite 同时还提供对每个帧的镜像翻转和90°倍数旋转的功能. LayerManager 是一个非常顺手的类,它记录你的场景中所有的层. 只需调用LayerManager的paint() 方法就可以绘制它所包括的所有层. 使用 TiledLayer TiledLayer 很简单, 尽管乍一看有些深入的地方不好理解. 最主要的思想就是一个图片源提供一组tiles (切片)用于组成一个大的场景. 例如, 下面的图片尺寸为64 x 48. 图片源 这个图片可以分为12个切片,每个大小为16 x 16 像素.
