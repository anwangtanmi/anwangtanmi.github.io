使用数码相机大约5年了吧，积累的照片文件（以及缩略图、短片）大约65G了！ 一直想把它们好好整理一下：把没价值、拍模糊了的那些，只保留一个低像素、低质量的缩略图就行了 能把总容量压缩到20G就好了。。。。。。。。 今天下载了picasa3来整理，但是好像没发现它把扫描过的目录里的照片的信息统一保存在什么地方？ 硬盘的剩余空间一直没怎么变。。。。。。 扫描完了，硬盘的各个分区的剩余空间还是没怎么变！ 莫非它真的不用保存缩略图到数据库的？？那怎么还会那么快？？？ 哦，是建了：C:/Documents and Settings/用户名/Local Settings/Application Data/Google/Picasa2/db3/ 大约990M，没感觉出来。。。。。。。。。。 哦，还在不断膨胀。。。。。。。。。会不会把我的系统盘搞满了？？！！ 没有自动把某些选择的照片替换为低像素、低质量的缩略图的功能 初步使用了一下：界面、功能的确很得心应手，大公司能把这样的小软件做得这么精致，的确不简单 可惜上传的空间只有1G。。。。。。。 其次，我想在浏览时，选择一些没价值、拍模糊了的照片，自动转为低像素、低质量的覆盖原来的大照片 以节省空间，但是它好像没遇到有人有这样的需求 那么退求其次：能允许我调用外部程序来实现？也不能扩展右键菜单 最后，只希望右键菜单里能有移动，把它们移动到一个目录，我使用自己的工具批量缩小它们，再一起移回来。但是右键菜单里也没有移动，只有删除——如果只是移动到回收站，也勉强够了。。。。。。。 ——哦，右键菜单里是有“移动到新文件夹…”的 还有：我希望在预览照片时，在左下方的空白处能现在照片的exif信息。也不行，只有在照片右下方显示直方图和相机信息，这个位置会覆盖照片，虽然半透明 还有，它的缩略图（索引数据）是保存在另一个目录的，如果把原图目录复制到另一台机器，这些索引数据也复制过去，能直接有效吗？

同一位最普通的商务人士。李彦宏如此低调，甚至从同机的一些记者身边经过时，都没有被认出来。等有些人认出他时，其身影已经消失在人流中了。 12月18日下午，李彦宏低调现身上海波特曼酒店，为百度上海研发中心揭幕。与李彦宏一起出现的是出任百度CTO仅两个月的李一男。李一男人称“技术天才”，曾是华为公司副总裁，后离开华为创业港湾网络公司。2006年，港湾被华为收购，李一男重回华为。2008年10月，履行完收购协议中规定的两年任期，李一男加盟了百度。 尽管李一男的传奇色彩吸引了一部分视线，但镁光灯频频聚焦的还是李彦宏。因为在过去的一个多月中，这家搜索引擎公司遇到了成立以来最大的难题，外间一直期待李彦宏能对此“有所表态”。 搜索引擎商业模式的争议由来已久，近期，央视连续对百度和谷歌的曝光将这种争议推到高潮，在谷歌大声抗辩、称“被央视误伤”同时，百度立即开始整 改。而相伴而行的却是铺天盖地的讨伐声。从传媒到网络，百度人人喊打，“再也不用百度了”的声音甚嚣尘上。此时，舆论已经一边倒，李彦宏再怎么解释都无济 于事。于是，他开始沉默，并淡出媒体视野。 在李彦宏“知错能改、善莫大焉”的道歉中，百度快速、“大力度”地出台了一系列整改措施：下线相关关键词、开除协助作假的员工、规范竞价排名。此外，12月初，原环球唱片中国区总经理梁康妮加盟百度，准备展开一系列“绿色新政”，正视一直饱受争议的音乐版权问题。 处于漩涡当中的李彦宏在上海的会场上，显得格外低调和安静。在演讲中，除了上海研发中心开幕一事之外，他还强调了搜索的未来和百度的优势，给公众传递更多的信心。 采访中，李彦宏对时代周报记者强调，百度会对竞价排名做出重大调整，更注重用户的搜索体验。 但不管怎么调整，都希望百度能坚持自己是个中文搜索引擎公司的原点。而此次百度高调进行上海研发中心的揭幕，并推出阿拉丁平台，就是一次向搜索技术 的极限难题发起挑战努力。李彦宏认为，阿拉丁平台可以解决现有搜索无法抓取和检索互联网上存在的大量hidden web（暗网）信息问题，并对人类的Hidden Knowledge（未知的知识—百度解释）进行挖掘和整理。为此专门挖来重磅搭档。 出现在上海的李一男，也是他加盟百度后在公众场合的首次公开露面，被业界视为技术天才的他接受李彦宏邀请执掌百度技术方略以来，人们一直想知道，他到底能为百度带来什么。 采访中，李一男表示：“如何将已知和未知信息分门别类地融合，有序地纳入到我们的知识范畴，并且能够为用户提供零成本、无障碍、无时差的精确送达，是有史以来我们在技术上面临的最大挑战。未来，百度将加大在研发上的投入，继续深化搜索技术。” 李一男强调，一直以来，搜索技术都是百度生存和发展的根本，百度在技术研发上的人力和资金投入，一直都是业内最多的。 受李彦宏之邀前来参加揭牌仪式的互联网技术专家高卢麟认为，随着百度对技术进一步强化，李一男在百度的作用也将更加明显。 一位业内资深人士认为，百度所推出的阿拉丁平台，“希望做一次曲线公关，向外界展示其战略调整的动向，其更大的意义则在于，百度重打技术牌，希望新的搜索技术能像阿拉丁神灯一样照亮用户的心，重拾用户对百度的信任。” 反思公司价值 指数 ( Indexes ) 今日焦点： · 投资秘密提供给美国机构 证监会27号函引争… · 中国股市遭遇历史性机会：股改行情大揭密 …(图文) · 中金“内部观点”曝光：表面唱多 暗里看空… · 炒股秘诀：10万赢100万 · 巴菲特–公开3只股票图 · 新办法炒股赚钱有新招 · 9:30公开2只涨停板黑马 证券市埸的相关指数是股票初入门者所必须注意观察的。以下是三种证券市场的平均数供投资者了解: 道琼平均指数 ( Dow Jone’s Average ) 道琼工业平均指数 (DJIA) 是最广为人知，也是最普遍被市场报导的指数。当有人说〝今天市场价又上涨了几个点数”，其实指的就是〝道琼〞上升几点。其计算方法已演变到非单纯的把所有公司的股价加总，在除以总公司的数目，现今的计算方法是取一特别值作为除数来计算其指数。道琼工业平均指数与现今大部份其余的指数不同，道琼工业平均指数并非以股票市价来计算，换言之，每天每只成分个股上涨的点数均需依照其一定的比重来换算，如此一来，每只成分股对指数大盘的影响力各有所不同。 道琼分为四种平均指数: 1.道琼工业平均指数 ( Dow Jone’s Industrials Average (DJIA) ) => 30种主要制造业的平均数。 2.道琼运输事业平均指数 ( Dow Jone’s Transportation Average (DJTA) ) => 20种交通事业股的平均数。 3. 道琼公用事业平均指数 ( Dow Jones Utility Average (DJUA) ) => 15种公用事业股的平均数。 4. 道琼65综合平均指数 ( Down Jones 65 Composite Average (DJCA)) => 综合以上三种65家公司的平均数。 一般而言，道琼工业平均指数的计算方法是以1897年（100％）为基准。在2000年2月25日道琼指数为10,092.
