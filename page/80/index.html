go用smpt包发送邮件, 被抄送收不到邮件bug

2019-01-29
今天在写一个比较简单的需求: 发送邮件且支持抄送 当时上网搜了一下, 直接用了 smpt 包来发邮件 网上介绍发邮件直接加 “Cc” 字段就可以实现抄送功能了, 但当我测试测试时. 结果是现在邮件上正确显示抄送人了, 但实际上被抄送人并没有收到邮件, 只有收件人收到了. 排查了会发现问题出在 Rcpt 操作上. Rcpt操作用于标识该电子邮件的计划接收人, 可以有多个RCPT行, 表示发送给多人. 之前我只对收件人邮箱地址进行了Rcpt操作, 而没有对抄送人邮箱地址Rcpt, 所以导致被抄送人收不到邮件.

暗网上出售的数据都来自哪里？

2019-01-28
互联网，被公开访问的网站，我们称之为“明网”。但还有非公开访问网站，我们统称他们为暗网和深网，暗网对于很多人来说，或许之前从没接触过，但在上面你的个人信息也许被明码标价出售。这也是法律难以打击而且暴利的交易活跃之地。 近日，腾讯安全对外发布《信息泄露：2018企业信息安全头号威胁报告》报告显示，暗网成为信息贩卖的主要渠道。这是2018年在暗网中交易的情况。 从数据分析来看，账号/邮箱数据、个人信息占比最高，分别为19.78%、12.19%，说明这些数据不但容易获取，并且也最受“欢迎”，同时企业信息类也在榜上，占比5.16%。 我们更关心的是这些数据是怎么泄漏在暗网上的？ 不法分子会利用技术攻击、钓鱼攻击和勾结内鬼、撞库攻击等手段获取大量一手信息，除了在暗网上直接售卖，也会用于精准诈骗、勒索等方式获利。 同时，我们不可避免的需要经常填写大量个人信息，用于注册、登记、邮寄等，这些信息会储存在对应的航空公司、酒店、科技公司、物流公司等系统的数据库里面，这些公司对数据管理的安全程度其实并没有我们想象那么高，其中很多公司对数据安全的重视程度非常低，甚至有核心数据不加密存储等低级错误。对于经验丰富的攻击者，弄到这些数据并不算难事。 3000万条的陌陌数据在暗网上出售； 1.3亿人开房记录被盗,放到暗网上拍卖； 至少5000万Facebook登录信息在暗网上销售； 华住集团的5亿条住客数据在暗网上打包售卖； 圆通10亿快递信息泄露暗网售价1比特币； 顺丰3亿用户数据在暗网出售； 14亿邮箱明文密码泄露惊现暗网。 其实这些都只是暗网交易中的冰山一角，这些可能多以技术性攻击导致，但也有诸多情况“内鬼”就充当了主角。 据权威数据统计，81%的政府与事业单位的泄密问题发生在体系内部(内部人员过失泄密或主动窃密)，由外部黑客攻击、系统漏洞、病毒感染等问题带来的信息泄密案例，合计仅有12%;内部体系造成的泄密损失是黑客攻击的16倍，是病毒感染的12倍。 由此可见，传统的网络安全在数据防泄密的措施中，依靠部署“防火墙”与“杀毒软件”等产品无法全面保障政府内部信息安全，政要部门的数据安全，已成为当前各级政府机关和事业单位亟待解决的安全隐患。

ubuntu16.04安装完成后重启，登陆之后卡死，进不去系统

2019-01-27
以下内容参考博文：https://blog.csdn.net/qq_31707275/article/details/82709133 解决方法： 1、重启，跳出grub界面后选择ubuntu，然后按e 2、在splash后面加nouveau.modeset = 0 3、按F10，输入登陆密码进入系统 4、打开终端，输入如下命令： sudo update-grub sudo gedit /etc/default/grub 5、找到GRUB_CMDLINE_LINUX_DEFAULT=”quiet splash”将其改为GRUB_CMDLINE_LINUX_DEFAULT=”quiet nouveau.modeset = 0″，保存并退出 6、更新 sudo update-grub 7、重启电脑 reboot

暗影精灵3 移动硬盘安装Ubuntu （Win + Linux）

2019-01-27
linux
暗影精灵3 移动硬盘安装 Linux系统 最近在学Linux，之前虚拟机用着很不爽，正好笔记本换了个SSD，于是把换下的128G硬盘买了个硬盘盒搞了个移动硬盘，双硬盘+双系统 岂不是美滋滋。 一. 安装前资源下载 WIN10安装很简单，网上随便一搜结果多的是，这里就不叙述了。接下来说一下在安装Linux系统前我们需要做的一些准备工作（我选择的Linux是Ubuntu的）： 准备一个大于等于4G的硬盘； 下载Ubuntu镜像文件，我是去官网下载的：https://www.ubuntu.com/download/desktop； 下载UltraISO软碟通，官网：https://cn.ultraiso.net/xiazai.html； 下载DiskGenius，http://www.diskgenius.cn/download.php； 二. 安装前资源配置 制作U盘启动盘： （1）安装完成打开软碟通软件 （2）【文件】-【打开】，打开下载好的Ubuntu镜像文件 （3）选中用作启动盘的U盘 （4）【启动】-【写入硬盘映像】（5）然后[写入]就OK了 2. 接下来对移动硬盘进行处理： （1）打开DiskGenius。注：这软件在我电脑上经常卡顿，不知是我下载的版本有问题还是怎样，总之在处理过程中多多等待别心急并不会影响使用 （2）因为之前这张盘我是用来装系统的，还没格式化，这里先将它格式化，如果之前有分区，需要把分区都删除才能格式化 （3）看该硬盘是否为GUID格式，如果不是转一下 （4）然后对硬盘进行分区，这里我是这样分的：/boot = 2G， /swap = 8G， / = 50G， /home = 剩余空间，可以根据不同的需求来进行分区，当然最简单的还是分一个 / 和一个swap。具体的操作就是点【新建分区】然后在出来的对话框中修改大小以及格式，格式的话除了swap选择 [Linux swap…] ，其他的都设置成 [EXT4] 即可。GPT不用区分主分区之类的，所以那一项不用管。 因为我的硬盘全都分完了，所以接下来的图去官网随便找张做示范了。 3. 分区完成，准备工作就已经全部OK了，接下来开始安装。 三. 安装Linux系统 由于昨天安装系统时忘了拍照，且安装并不是很麻烦，这里根据记忆进行叙述： 将U盘与移动硬盘都连接到电脑上，开机并按 [F9]，选择从U 盘进行启动； 然后选择第二项 [ Install Ubuntu ] ; 开始是一些常规设置，如语言、完全安装还是精简安装之类的。要注意到了安装类型那一步，选择最下面的 【其他选项】； 因为做准备工作时已经将移动硬盘进行了分区了，接下来就是找到那几个分区，进行挂载。记得没错的话是每点到一个分区下方会有个【更改】选项，只需要在弹出的对话框中将【用途】那一栏选一下就OK了，4个分区都要配置一下（这里是因为我的分区方案是分为4个）； 可以看到下方还有个引导设备安装位置的选项，这里我选择的是移动硬盘（关于这个选择有点小问题，后面会提到）； 到了这里可以说系统就等于装好了，点击安装，接下来还有些常规设置，需要注意的可能也就是别设置了密码自己回头忘掉。 四. 你以为这就结束了吗？hhhhhhhhh 系统安装后所遇问题汇总： 当你安装完成，输好密码，然后画面卡住不动了，该怎么办？ 这里我遇到了这种情况，因为暗影精灵3是双显卡，自带的N卡跟Ubuntu自带的显卡驱动存在一些冲突，所以每每输入密码准备登录时都会卡住。 解决办法是在开机时选择recovery模式进入Ubuntu，然后参照这个帖子进行设置。https://blog.csdn.net/muyuxiaoxiang/article/details/80502703
