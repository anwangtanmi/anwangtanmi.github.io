持续走高的比特币将在2016年降低？

2016-04-18
2016即将见证比特币价格的又一次上涨。其背后的原因不是所谓的暗网毒品交易也不是俄国的庞氏骗局，而是一个相对不那么耸人听闻的因素，即货币供应增长的减缓。 比特币是一种基于网络的虚拟货币，不依靠中央机构存在，为用户提供一种快捷、匿名的货币流通方式。尽管受到一些人的推崇，认为其是未来货币的模板，还是存在一些人认为它不够安全不适合以投资货币的模式来使用。 2016与众不同的原因在于，比特币和传统法定货币一样受到相似因素的影响，经久不变的供需因素影响了这个新型货币。 比特币不受中央银行的控制，而是依靠一个所谓的‘挖掘’计算机系统存在，这种系统会每十分钟一次竞先通过数学运算来完成交易区块的验证。作为回报，最 先算出结果，清空交易的会受到25新比特币的奖励，价值约11,000BTC=BTSP。 但是当这种机制于2008年首次被一位化名为‘中本聪’的人发明出来时，第一版机制是每四年奖励额度就会减小一半以防止通货膨胀的出现。下一次四年正是2016年7月。 比特币也像普通商品一样，限量发行2100万，目前比特币发行有1500万，在未来的125年有望达到这个数字。因此用‘挖掘’来形容非常贴切。 设立于泽西的环球百万比特币对冲基金的联合创始人Daniel Masters最初在19世纪80年代只是是壳牌公司的一个石油交易商，30年营销货物的经历，最终他遇到了比特币。 现在他认为比特币价格明年有望超过2013年达到1100美元，接着还会继续保持涨势，2017年预计会达到4400美元。 Master说：“这背后自然有许多因素的影响，包括大型公司机构对比特币支付模式的接受，‘模块链’技术中迅速增长的投资和利润（模块链是支持比特币运营的技术）还有随着货币贬值经济增长缓慢导致的中国需求量的增长。 但是Master认为把比特币单独拿出来看，奖励机制的减少将会造成其价格约50%的增长。就像在其他市场中一样，奖励机制的减少已经是不可避免的现实。 这就像如果OPEC（石油输出国组织）明天能够站出来说‘石油价格将会在六个月内有所下降石油价格就会立刻受到影响一样。比特币还处于发展初期，而且我不认为这个因素将会完全影响到价格中，Master说。 不受统一控制的虚拟资产 在过去的三个月里比特币价格几近翻了一番，在过去两年中这是增长最快的一个季度。由于中国对金字塔计划的需求（这个金字塔有俄国的欺诈者建立，是价格上涨的部分原影音），自去年八月以来，上个月首次达到了500美元。 中国比特币交易公司BTCC的首席执行官 Bobby认为虚拟货币还有很长的路可以走。他认为在2016年7月之前价格还能增长8倍，明年夏天会达到3500美元。 Lee说：“考虑到流通中的比特币共有65亿美元，今天比特币的价格约为世界人均1美元。 2012年11月奖励机制就已经经历过一次下降，从50比特币下降到25比特币。从那之后股份就更低了，一比特币价值约为12美元，然而在接下来的7个月里，价格又增长了150%，近乎到了下一次降值的时间。 来自伦敦金融技术研究中心Magister Advisors的Jeremy Millar说：“其他的影响都不大，但降值严重影响了供应，这为比特币价格带来了很大的压力。”他期望需求量能够继续增长。 “没有人会质疑这个基本的经济原则。”

笔记本装 Win10 后的亮度调节和 WiFi 问题

2016-04-17
网络安全
笔记本之前用的 Win7 是盗版的，日常使用没什么大的问题，但是系统字体怎么设置感觉都不够美观，甚至不如另一台 XP 上的字体好看。因此去年趁着 Win10 推广的浪潮，升级到了 Win10，一是为了体验口碑不错的 Edge 浏览器，二是如果系统体验还不错就可以顺带洗白了，体验不好就回滚到 Win7。本来图方便想使用 360 的一键升级助手，但在检测时提示硬盘检测未通过，只能通过微软的官方升级助手升级，也比较简单。 装好 Win10 后，系统字体一下子清爽了许多，测试了一下其它之前安装的程序，大部分都没问题，使用正常，极个别的需要重新安装。这一体验就到了一个月之后，主观和客观上都回不去 Win7 了，遂决定以 Win10 为主力。为了清理系统和提升性能，索性从 MSDN 上下载了镜像进行重装，再用驱动人生更新驱动，一切顺利，没发现什么问题。但当时没发现问题不代表就没有问题，只是时间不够，局势没搞清楚。 亮度调节 因为通常使用都是插上电源的，屏幕亮度也合适，从没调过，但有一次用电池时发现屏幕很暗且亮度无法调节。于是去网上查方法，试验，再查，再试验，循环交替。终于，找到了一个可用的方法，记录备用。 快捷键 Win+R，打开“运行”，输入services.msc回车，在本地服务中禁用Sensor Monitoring Service服务； 快捷键 Win+R，打开“运行”，输入regedit进入注册表，点击菜单项中的“编辑”-》“查找”，搜索KMD_EnableBrightnessInterface2，然后右键“修改”，将里面的“数值数据”从 1 改为 0； 重启，试试亮度调节吧。 我的显卡是 AMD HD 5000 的，更新后亮度调节完美。如果后面自行更新 AMD 驱动的话，需要重新到注册表再改一下。 WiFi 不稳定 使用 Win10 后的另一个问题就是 WiFi 变得不稳定，时不时就连不上网了，重启下又好了，过会又不行了。每次网络变慢时网页解析时间都很长，还以为是 DNS 的问题，换过 114，阿里和中科大的 DNS，来来回回折腾不知多少次，都不行。再一次翻阅浩瀚的网络，果然有不少人反馈，方法也多种多样，再次循环测试搞定了问题。 首先，在电脑右下角的网络图标上右击，弹出菜单后单击“打开网络和共享中心”，选择“更改适配器设置”，右击连接的 WiFi 网络，依次打开“属性”-“配置”-“高级”-“Wireless Mode”, 值选择为”IEEE 802.11b/g“； 同时，也可以根据 WiFi 信号的强弱，设置下“Roaming Sensitivity Level”，如果值设为”High“，电脑的接收灵敏度会有所提高，但是在网络信号好的地方建议还是设为”Low“，因为会影响网速。 最后，右键单击“开始”键，点击“设备管理器”，打开后找到“网络适配器”，选择有问题的无线网卡，右键选择“属性”，切换到“电源管理’的选项，取消勾选“允许计算机关闭此设备以节约电源”，确定保存，然后重启电脑。 目前使用 Win10 系统总体都还不错，但在打开一些大型软件（如：PhotoShop）时，和 Win7 比还是会有些卡，如果是工作用机暂时不建议更换 Win10。

2016-04-14
现在，网上购买黑客服务比以往便宜多了。 《华尔街日报》报道称，戴尔旗下安全公司SecureWorks最新报告显示，情报分析人员发现俄罗斯及其他黑客的地下雇佣市场出现飞速增长的趋势。 经过近八个月对互联网上数十个暗网的调查，分析人员表示，包含括病毒和其他捆绑软件的恶意软件变得“更加便宜，并且大大降低了黑客入侵的门槛”。根据调研报告，新黑客教程，比如“如何发送一封钓鱼邮件”等，可以在网上以20到40美元不等的价格购买得到；计算机远程访问“木马程式”一类教程，甚至低至5至10美元。 此外：黑客对从Gmail或者雅虎账号盗取个人邮件要价129美元，盗取企业电子邮件账户的要价则要高得多——每个邮箱收取500美元。报告并未揭露这些在线黑客交易的承诺兑现情况，但里面指出，某非法服务提供者自称，他们能够神不知鬼不觉地盗取受害者的邮件，而不让人发现任何一点可疑。 此报告的发表正值执法者对这个新兴的黑客雇佣市场“hacker-for-hiremarket”给予越来越多的关注之际，这个地下雇佣市场允许任何有条件接入互联网以及能够支付一点佣金的人轻易地就对互联网安全带来破坏。黑客攻击已经成为催生了各种各样犯罪事件的行为，包括非法赌博、内部交易等等。 报告称，黑客业务的操作已经趋向于一般的企业，比如许多俄罗斯黑客都会提供24/7——“一天24小时，一星期7天”的服务。分析人员还发现，黑客兜售他们的产品时像足一个典型的创业公司——比如打出“免费攻击试用”、“强大的攻击能力”等广告。 在暗网上，信用卡号码、银行账号凭证、护照等身份盗用仍是最受欢迎的标配产品。此外，DellSecureWorks发现，黑客现在也开始兜售常旅客账户和酒店分账户，而这一趋势此前《华尔街日报》已有所报道。据悉，这些账户能够用来在合法网站上兑换礼品卡。
